css语法-伪元素失效

什么是伪元素

表示页面中一些特殊的并不真实的存在的元素（特殊的位置），本质是一个元素的子元素
使用伪元素时，要特别注意它只能用于某种元素，以及只能应用某些属性

  • ::first-letter 用于向文本的首字母设置特殊样式

    • 只能用于块级元素。

      首字样式

  • ::first-line 用于向文本的首行设置特殊样式。

    • 只能用于块级元素。

      这是一段不止一行的文字这是一段不止一行的文字这是一段不止一行的文字这是一段不止一行的文字这是一段不止一行的文字这是一段不止一行的文字这是一段不止一行的文字这是一段不止一行的文字

  • ::selection 表示选中的内容

  • ::before 在元素的内容前面插入新内容。

    • 必须配合 content 使用
    • img、input 和其他的单标签是没有 after 和 before 伪元素的

  • ::after 在元素的内容之后插入新内容。因为单标签本身不能有子元素（比如 img 标签里不能再包裹内容）

    • 必须配合 content 使用

伪元素失效的原因

  • 未写 content
  • 应用于单标签

伪元素与伪类的区别

  • 伪类选择元素基于的是当前元素处于的状态，或者说元素当前所具有的特性，而不是元素的 id、class、属性等静态的标志。

    由于状态是动态变化的，所以一个元素达到一个特定状态时，它可能得到一个伪类的样式；当状态改变时，它又会失去这个样式。由此可以看出，它的功能和 class 有些类似，但它是基于文档之外的抽象，所以叫伪类。

  • 伪元素是对元素中的特定内容进行操作，因此它的动态性比伪类要低得多。

    设计伪元素的目的就是去选取诸如元素内容第一个字（母）、第一行，选取某些内容前面或后面这种普通的选择器无法完成的工作。它控制的内容实际上和元素是相同的，但是它本身只是基于元素的抽象，并不存在于文档中，所以叫伪元素。
